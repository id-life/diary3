# Diary3 项目概述

## 项目基本信息

| 属性 | 值 |
|------|-----|
| **项目名称** | Diary3 |
| **版本** | 0.1.195 |
| **框架** | Next.js 15.1.11 (App Router) |
| **语言** | TypeScript 5.1.6 |
| **包管理器** | pnpm 10.0.0 |
| **状态管理** | Jotai 2.12.5 |

## 项目功能简介

这是一个个人**日记/习惯追踪应用**，主要功能包括：

1. **习惯管理** - 创建、编辑、删除习惯类型 (EntryType)
2. **每日记录** - 记录每日习惯完成情况 (EntryInstance)
3. **数据可视化** - 图表展示习惯完成统计
4. **连续记录追踪** - 计算和展示习惯连续完成天数 (Streaks)
5. **提醒功能** - 设置习惯提醒 (ReminderRecord)
6. **GitHub OAuth 认证** - 用户登录和数据云备份
7. **数据导出** - 导出数据到文件或数据库

## 技术栈概览

### 前端框架
- **Next.js 15** - React 全栈框架，使用 App Router
- **React 19** - UI 库
- **TypeScript** - 类型安全

### 状态管理
- **Jotai** - 原子化状态管理（主要）
- ~~Redux~~ - 已完成迁移，不再使用

### UI 组件库
- **Radix UI** - 无样式组件原语
- **Ant Design** - 部分复杂组件
- **Lucide React** - 图标库
- **Tailwind CSS** - 原子化 CSS 框架

### 数据可视化
- **Recharts** - React 图表库

### 数据获取
- **TanStack React Query** - 服务端状态管理
- **Axios** - HTTP 客户端

### 表单处理
- **React Hook Form** - 表单状态管理
- **Zod** - 数据验证

### 动画
- **Framer Motion** - 动画库
